# 发布后的测试脚本 


## 本地网络 

### 创建池子-初始化价格-模拟交易-对账-移除流动性

```bash
npx hardhat run scripts/full-flow-test.js --network hardhat  
```

### 原子化一些脚本，拼接场景测试 对账是重点

```bash
# 5个用户，每个20笔交易
USERS=5 SWAPS=20 npx hardhat run scripts/scenarios/reconciliation-test.js --network hardhat

# 更多用户和交易
USERS=20 SWAPS=100 npx hardhat run scripts/scenarios/reconciliation-test.js --network hardhat

# 指定报告文件名
REPORT=my-test-report.json npx hardhat run scripts/scenarios/reconciliation-test.js --network hardhat

# 安静模式（减少输出）
QUIET=true npx hardhat run scripts/scenarios/reconciliation-test.js --network hardhat

# 组合使用
USERS=5 SWAPS=20 REPORT=quick-test.json npx hardhat run scripts/scenarios/reconciliation-test.js --network hardhat
```
